⚠️ PENDIENTE DE BACKEND
================================================================================

Para completar la integración, el backend debe implementar:

1. POST /api/reactions/toggle
   Body: { targetId: string, type: ReactionType }
   
2. GET /api/reactions/{postId}
   Response: Array<{ userId, username, reactionType }>
   
3. GET /api/search/posts?query={query}
   Response: Post[]
   
4. GET /api/search/hashtag?tag={tag}
   Response: Post[]
   
5. GET /api/hashtags/trending
   Response: Array<{ tag: string, count: number }>

NOTA: Las funcionalidades funcionan con estado local hasta que se 
implementen estos endpoints.
