ENDPOINTS DE BACKEND REQUERIDOS (PENDIENTES):
================================================================================

1. POST /api/reactions/toggle
   Body: { targetId: string, type: ReactionType }
   Response: { success: boolean }

2. GET /api/reactions/{postId}
   Response: { userId: string, username: string, reactionType: ReactionType }[]

3. GET /api/search/posts?query={query}
   Response: Post[]

4. GET /api/search/hashtag?tag={tag}
   Response: Post[]

5. GET /api/hashtags/trending
   Response: { tag: string, count: number }[]

================================================================================
CÓMO USAR LAS NUEVAS FUNCIONALIDADES:
================================================================================

1. HASHTAGS Y MENCIONES:
   - Escribe #hashtag en cualquier post
   - Escribe @usuario en cualquier post
   - Los hashtags y menciones se convertirán automáticamente en links
   - Click en un hashtag para buscar posts con ese hashtag
   - Click en una mención para buscar ese usuario

2. COMPARTIR:
   - Click en el botón de compartir (Share2 icon) en cualquier post
   - Selecciona una opción: copiar link, redes sociales o QR
   - El código QR se genera automáticamente

3. REACCIONES:
   - Mantén presionado el botón de like para ver el selector
   - Selecciona una reacción (❤️😂😮😢😡)
   - Click en el contador para ver quién reaccionó
   - Click nuevamente en la misma reacción para removerla
