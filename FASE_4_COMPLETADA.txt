1. POSTS EN GRUPOS:

   - Click en "Publicar en el grupo"
   - Escribir contenido y publicar

2. ROLES EN GRUPOS:
   - Tab "Miembros" en grupo
   - Admin puede cambiar roles
   - Botones "Hacer Moderador" / "Remover"

3. INFINITE SCROLL:
   import { useInfiniteScroll } from '@/hooks/use-infinite-scroll'
   
   const { loadMoreRef } = useInfiniteScroll({
     onLoadMore: fetchMore,
     hasMore: true,
     isLoading: false
   })
   
   <div ref={loadMoreRef} />

4. CACHÉ:
   import { cache, cachedFetch } from '@/lib/utils/cache'
   
   const data = await cachedFetch('key', () => fetch(...))
   cache.invalidate('key')

5. PWA:
   - Agregar en layout.tsx:
   import { usePWA } from '@/hooks/use-pwa'
   usePWA()
   
   - La app es instalable en móvil/desktop
